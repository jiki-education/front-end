title: "Using Functions"
description: "Calling functions to make things happen. The first step in programming."
---

Functions are one of the most fundamental concepts in programming. You shoud think o them like little machines that you can tell Jiki to use.

To use a function, you write it's name, followed by two brackets. For example, if you want to use a function called "move", you write `move()`. Similarly if you want to use a function called "turnLeft", you write `turnLeft()`.

Functions in JavaScript use camelCase. All the words in the name are pushed together, and we use capital letters to signify the start of functions.

In other languages like python we use underscores instead, so we'd write `turn_left()` not `turnLeft()`.
