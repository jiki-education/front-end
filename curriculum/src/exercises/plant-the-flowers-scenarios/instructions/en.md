title: "Plant the Flowers (Scenarios)"
description: ""
---

The gardener wants flowers planted evenly across the garden, but the number of flowers changes each time!

Use `numFlowers()` to find out how many flowers to plant. Then work out the gap between them so they're evenly spaced across the garden (which is 100 units wide).

The formula for the gap is: `100 / (number of flowers + 1)`.

For example, if there are 3 flowers, the gap is `100 / 4 = 25`, so the flowers go at positions 25, 50, and 75.

Use the `plant()` function to plant each flower at the right position.
