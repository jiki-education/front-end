Your task is to draw a weather icon for a rainy day with the sun peeking out behind the clouds. When finished, it should look like this:

<img src="/static/images/exercise-assets/cloud-rain-sun/finished.png" alt="Finished cloud, rain and sun scene" style="max-width: 300px; border: 1px solid rgba(0, 0, 0, 0.3); border-radius: 5px; box-shadow: 0 0 3px rgba(0, 0, 0, 0.1); margin-bottom: 8px;" />

We've drawn a template for you. Your shapes should sit just inside the lines. We've also added the base of the cloud into the code to make things a little easier. You can't change this line.

You'll need the `circle` and `ellipse` functions. You can find the full references below. You should only use ellipses for the raindrops. If you use them for the sun or clouds, things won't work.

Nearly all the numbers you need are divisible by 5 (e.g. 5, 10, 15, etc). The only exception is the horizontal radius of the raindrops, which is `3`.

Have fun!
